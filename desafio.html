Sobre Projeto

O Portal Jovem Programador reúne conteúdos para estudantes que estão começando a aprender HTML.

A missão é praticar estrutura, organização e semântica usando tags como header, main, section e footer.


Aprender HTML é o primeiro passo para criar páginas web.

Trilha de Estudos

Siga a ordem abaixo para montar a página:

  1. Criar a estrutura inicial com html, head e body.
  2. Adicionar cabeçalho, título principal e menu de navegação
  3. Organizar o conteúdo principal em seções semânticas.
  4. Inserir listas, imagem, tabela e links.
  5. Finalizar com rodapé e informações de contato.

Tags que devem aparecer

    Tags semânticas: header, nav, main, section e footer.
  • Tags de texto: h1, h2, h3, p, strong e em.
  • Tags de conteúdo: img, figure, figcaption, ol, ul, table e a

Dica da semana

Use tags semânticas sempre que possível. Elas deixam o código mais claro para pessoas, navegadores e leitores de tela.

Regras do Desafio

    Recrie a página sem copiar o código original.
  • Use indentação para organizar o HTML/li>
  • Confira se os links, imagens, listas e tabela aparecem corretamente.
  • Não precisa usar CSS nesta atividade.
